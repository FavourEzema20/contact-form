# Personal Portfolio & Registration Form Website

A simple HTML project with a homepage and contact form

## Project Overview

This project is a beginner-friendly website built using HTML5. It consists of a personal portfolio homepage and a registration form page. The website showcases personal information, hobbies, daily routines, favourite qoutes, and a user registration form.

The project demonstrates the uses of semantic HTML elements, forms, lists, embedded content, and navigation between pages.

## Features

### Home Page

1. Personal introduction section.

2. Hobbies list.

3. Daily routines.

4. Navigation link to contact page.

5. Embedded video content, Google Map, and YouTube tutorial.

6. FAQs.

7. Favourite quotes.

8. Footer section.

### Registration Form Page

1. Full Name input field.

2. Email input field.

3. Password input field.

4. Gender selection using radio buttons.

5. Multiple hobby selection using checkboxes.

6. Course selection dropdown.

7. About You textarea.

8. Navigation back to the homepage.

## Technologies Used

HTML5

## How to Run the Project

1. Download or clone the project

2. Open the project folder

3. Open index.html in your browser

4. Navigate through the website using the provided links
